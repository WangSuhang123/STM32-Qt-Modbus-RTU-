# STM32-Qt Modbus RTU 通信项目

基于 STM32F103C8T6 与 Qt 6.7 的双向串口通信系统，采用类 Modbus RTU 自定义协议。

## 硬件配置

| 组件 | 型号 | 说明 |
|------|------|------|
| 主控板 | STM32F103C8T6 | ARM Cortex-M3 |
| 通信模块 | CH340 USB转TTL | 串口通信 |
| 温度传感器 | Risym热敏传感器模块 | 环境温度采集 |
| 光照传感器 | 光敏电阻传感器模块 | 光强度监测 |
| 显示模块 | OLED显示屏 | 实时数据显示 |
| 报警模块 | 蜂鸣器 | 超限报警 |

## 通信协议

### 帧格式

| 字段 | 字节数 | 示例值 | 说明 |
|------|--------|--------|------|
| 帧头 | 2 | 0xAA 0x55 | 同步头，防止粘包/干扰 |
| 地址码 | 1 | 0x01 | 设备地址（可扩展多设备） |
| 功能码 | 1 | 0x01/0x02/0x03 | 命令类型 |
| 数据段 | 1~N | 可变 | 具体数据 |
| 数据长度 | 1 | 0x01 | 数据段实际长度 |
| CRC16 | 2 | 0xAB 0xCD | Modbus标准CRC-16（低字节在前） |

**总长度** = 2(头) + 1(地址) + 1(功能) + 1(长度) + N(数据) + 2(CRC)

### 功能码定义

#### Qt → STM32（命令）

| 功能码 | 名称 | 数据 | 说明 |
|--------|------|------|------|
| 0x01 | 开启光照监测 | 0x01 | 启动光敏传感器监测 |
| 0x02 | 关闭光照监测 | 0x00 | 停止光敏传感器监测 |
| 0x03 | 查询状态 | 无（长度=0x00） | 查询当前系统状态 |

#### STM32 → Qt（响应）

| 功能码 | 名称 | 数据格式 | 说明 |
|--------|------|----------|------|
| 0x81 | 状态应答 | [使能标志:1字节][光敏值:1字节] | 返回当前监测状态和传感器数据 |

### 协议示例

**开启光照监测请求：**

## CRC16校验

采用Modbus标准的CRC-16校验，校验范围覆盖帧头至数据段所有字节。

**C语言实现:**

```c
uint16_t Modbus_CRC16(uint8_t *data, uint16_t len) {
    uint16_t crc = 0xFFFF;
    for (uint16_t i = 0; i < len; i++) {
        crc ^= data[i];
        for (uint8_t j = 0; j < 8; j++) {
            if (crc & 0x0001) {
                crc = (crc >> 1) ^ 0xA001;
            } else {
                crc >>= 1;
            }
        }
    }
    return crc;
}
```

## 使用说明

### 1. 硬件连接

```
STM32F103C8T6          CH340 USB转TTL
----------------       ----------------
PA9 (TX)      ───────►  RX
PA10 (RX)     ◄───────  TX
GND           ───────►  GND

光敏传感器模块:
PA0 (ADC)     ◄───────  AO输出

OLED显示屏 (I2C):
PB6 (SCL)     ◄───────  SCL
PB7 (SDA)     ◄───────  SDA

蜂鸣器:
PC13          ◄───────  信号输入
```

### 2. Qt上位机操作

1. 选择正确的串口号
2. 设置波特率为115200
3. 点击"连接"按钮
4. 使用控制按钮发送命令
5. 观察实时数据与图表

### 3. STM32固件烧录

1. 使用ST-Link或J-Link连接开发板
2. 打开Keil工程并编译
3. 下载固件到STM32F103C8T6

## 扩展说明

当前协议设计支持多设备扩展：

- **地址码 0x01~0xFF**: 可连接多个从机设备
- **功能码自定义**: 可根据需求扩展温度监测、湿度监测等功能
- **数据段可变长**: 支持不同类型命令携带不同长度的数据
